CARACTERISTICAS Conoce ARAMBI!

ARAMBI

ARAMBI es un addin de Autodesk Revit creado con el objetivo de facilitar las tareas de presupuestar y planificar proyectos de edificación con un interfaz sencilla y amigable.

¿Porque presupuestar sus Proyectos con
ARAMBI?

ARAMBI trabaja directamente con su modelo BIM en Autodesk Revit y permite enlazarlo con una base de dato propia generando un presupuesto de manera automática.

¿Porque planificar sus Proyectos con
ARAMBI?

ARAMBI tiene un interfaz intuitivo para planificar su proyecto en función a la producción.

ARAMBI
Metrado!

ARAMBI obtiene los metrados de concreto y encofrado con el uso de dynamo. Ademas tiene una base de datos modificable para obtener el metrado de acero mediante ratios.

ARAMBI
Presupuesto!

ARAMBI elabora un presupuesto mediante la elección de partidas con sus respectivos APU's y una lista modicable de recursos. El presupuesto se obtiene disgregado en Materiales, Mano de Obra, Equipo/Herramientas y Subcontrata

ARAMBI
Planificación!

ARAMBI tiene el potencial de asignar el tiempo de ejecución de cada elemento del modelo mediante una sectorización y la generación de un tren de actividades mediante un interfaz amigable
